# ダッシュボード

> URL: http://localhost:3000

ログイン後に最初に表示される画面です。記事生成の全体状況を一目で確認できます。

---

## 画面構成

### ナビゲーションバー（上部）

| ボタン | 説明 |
|--------|------|
| **ダッシュボード** | この画面に戻る |
| **成果物** | 完成した記事の一覧 |
| 🌙/☀️ ボタン | ダーク/ライトモード切り替え |
| **Settings** | 設定画面へ |
| **+ New Run** | 新しい記事作成を開始 |

### 統計カード（5枚）

画面上部に表示される5つのカードで、現在の状況を数値で確認できます:

| カード | 説明 |
|--------|------|
| **総実行数** | これまでに作成した記事生成の総数 |
| **完了** | 正常に完了した数 |
| **進行中** | 現在実行中 + 承認待ちの数 |
| **失敗** | エラーで停止した数 |
| **成功率** | 完了数 ÷ 総実行数（%） |

### 実行履歴（下部）

過去の記事生成（Run）が一覧表示されます。各行をクリックすると、そのRunの詳細画面（ワークフロー進捗）に遷移します。

#### 表示される情報

- **キーワード**: 記事のターゲットキーワード
- **ステータス**: 実行中 / 承認待ち / 完了 / 失敗
- **進捗**: どの工程まで進んでいるか
- **作成日時**: Runを作成した日時

→ 詳しくは [02_新規記事作成.md](02_新規記事作成.md) を参照

ナビゲーションの **「成果物」** をクリックします。
→ 詳しくは [04_成果物閲覧.md](04_成果物閲覧.md) を参照
