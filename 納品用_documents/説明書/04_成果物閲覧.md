# 成果物閲覧

> URL: http://localhost:3000/articles

完成した記事の閲覧と管理を行う画面です。

---

## 記事一覧画面

ナビゲーションの **「成果物」** をクリックすると、完成した記事の一覧が表示されます。

### 表示される情報

| 項目 | 説明 |
|------|------|
| **キーワード** | 記事のターゲットキーワード |
| **記事タイプ** | メイン / 初心者向け / 実践編 / まとめ |
| **生成日時** | 記事が生成された日時 |
| **ステータス** | 完了 / レビュー中 等 |

---

## 記事詳細画面

一覧から記事をクリックすると、詳細画面が表示されます。

### 閲覧可能な形式

| タブ/ビュー | 説明 |
|-------------|------|
| **Markdown** | マークダウン形式のプレビュー |
| **HTML** | WordPress向けHTML形式 |
| **JSON** | 構造化データ（メタ情報含む） |

### 記事内容

各記事には以下の要素が含まれます:

- **タイトル**: SEO最適化されたタイトル
- **メタディスクリプション**: 検索結果用の要約
- **見出し構成**: H2/H3/H4の階層構造
- **本文**: SEO最適化された記事本文
- **FAQ**: よくある質問と回答
- **CTA**: 設定したCall To Action

---

## プレビュー機能

記事の最終的な見た目を確認できます。

- **HTMLプレビュー**: 実際のWebページに近い表示
- **WordPress形式**: Gutenbergブロック形式でのプレビュー

---

## GitHub連携（設定時）

GitHubリポジトリと連携している場合、記事をGitHub上にPush/PRとして送信できます。

- **GitHub Actions ボタン**: 記事詳細画面で表示
- 連携の設定は [05_設定画面.md](05_設定画面.md) のGitHubタブを参照
